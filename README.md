# Skills

This repository contains Agent skills — reusable prompt-based commands that extend coding-agent's capabilities. Skills let you define specialized behaviors, automate workflows, and invoke complex tasks with a simple `/skill-name` command.

## Usage

To use a skill, use [`gh`](https://cli.github.com/) or [`npx skills`](https://skills.sh/).

```bash
gh skill install rmuraix/skills --agent claude-code
```

```bash
npx skills add rmuraix/skills
```
